# Lukebarton Local

## How do I install these formulae?

`brew install lukebarton/local/<formula>`

Or `brew tap lukebarton/local` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lukebarton/local"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
